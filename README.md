#Журнал вихідної, вхідної та вхідної ел. пошти
==============================================

###Журнал в розробці
--------------------

На даний час працює вихідна кореспонденція та журнал бланків.

###Наступні кроки
-----------------

  - Система пошуку та швидкий пошук.
  - Відновлення з копії БД.
  - Вхідна кореспонденція.
